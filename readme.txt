## Instruções:

1-Instale o python e o runtime java.
2-Instale o vscode ou o pycharm community.
3-Crie uma nova pasta "PDF_CLARO" e dentro dela crie outra pasta "pdf".
4-Dentro da pasta raíz "PDF_CLARO", adicione o script main.py e o requirements.txt.
5-Execute o editor de código e abra a pasta do projeto "PDF_CLARO".
6-Coloque os pdfs na pasta "pdf".
7-Execute o script.

## Se todos os requesitos e passos foram seguidos corretamente será gerado um arquivo "report_claro.xlsx" dentro da pasta raíz "PDF_CLARO".
